# 技能：種族/陣營詞條撰寫 (extract_factions)

**用途**：根據原始素材與 `entities.md` 名單，撰寫種族與組織陣營詞條，輸出至 `results/<作品>/factions.md`。重要陣營若內容超過約 25 行，拆出 `results/<作品>/factions/<name>.md`。

---

## 提示詞

根據以下原始素材與 `entities.md` 中的「種族/陣營」分類，為《{作品名稱}》撰寫詞條。以繁體中文輸出。

### 鐵則

- **窮盡名單**：所有具名種族（人類、精靈、矮人……）與組織陣營（國家、教團、傭兵團、黑幫、政黨）都要有條目。
- **短條目優先**：3–8 行為默認。文化複雜、歷史悠久者才展開。
- **長條目拆檔**：超過約 25 行拆到 `factions/<name>.md`。
- **全劇透**：陣營的真實意圖、幕後操縱者、最終命運（被滅、變質、勝利）都寫。
- **媒介翻譯**：陣營若有標誌性視覺符號（旗幟、儀式、建築風格），轉述其敘事意義（如「他們的戰旗用倒掛的聖徽，呼應原教義已被顛覆」）。

### 詞條結構（彈性）

```
### {名稱}（{原文}）

> {一句話定位：種族 / 國家 / 組織 / 教團 / ...}

- **起源**：{從何而來、由誰建立、誕生背景}
- **構成**：{種族、規模、組織形式}
- **目標 / 信念**：{他們想要什麼、相信什麼}
- **與其他陣營的關係**：{盟友、敵人、附庸，連結到對方詞條}
- **關鍵人物**：{領袖、代表性成員，連結到 characters 詞條}
- **故事中的角色**：{推動了什麼、阻礙了什麼、結局}
- **敘事象徵**（可選）：{這個陣營在作品中代表什麼概念}
```

### 排序

按敘事重要性排序，**而非按遊戲強度或人口數量**。

**輸出格式**：Markdown。
