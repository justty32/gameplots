# 瑪娜希斯回響 - 萌娘百科原始資料 (繁體中文轉譯)

## 1. 劇情與世界觀詳情
*   **背景傳說**：在久遠的時代，精靈（瑪娜希斯）之間爆發戰爭，誕生了名為「魔王」的邪惡存在。世界瀕臨毀滅之際，寄宿著瑪娜希斯之力的勇者及其夥伴擊敗並封印了魔王。
*   **現狀**：魔王雖已消失，但殘留的力量仍困擾世人。為了防止魔王復活，勇者們在各地建立了培育後進的學院——「西奧德里克學院」。隨著和平持續，人們漸漸遺忘魔王，學院的主要工作轉向研究瑪娜術法與培育冒險者。
*   **主線劇情**：玩家因意外事故被任命為「箱庭島」分院學院長的助手，開始與少女們在學院學習冒險知識、前往各地採集素材並解決問題的青春冒險日常。

## 2. 具名人物清單及其背景、語音、喜好

### 主要少女（學生）
1.  **塞納菲娜·阿什克羅夫特 (Serafina Ashcroft)**
    *   **背景**：來自東方精靈村落，因幼年被法師相救而立志成為法師。
    *   **語音 (CV)**：石見舞菜香
    *   **喜好/特徵**：行動派，擅長料理與打掃，喜歡小巧可愛的物品。對矮小身高有些自卑。
2.  **米爾德里德·彌爾蒂 (Mildred Mirti)**
    *   **背景**：出身南方港灣都市里卡薩的劍士家族，為了成為一流冒險者而入學。
    *   **語音 (CV)**：下地紫野
    *   **喜好/特徵**：正義感強、認真。喜愛甜食（但會克制），討厭辛辣與燙口的食物。
3.  **吉爾伯塔·梅普爾 (Gilberta Maple)**
    *   **背景**：孤兒，由瑪娜希斯教會撫養長大，為了報恩選擇牧師職位。
    *   **語音 (CV)**：關根瞳
    *   **喜好/特徵**：面無表情，喜歡收集形狀奇妙的物品。能平靜地吃下超辣食物。
4.  **法露露·卡普安納 (Faruru Capuana)**
    *   **背景**：森之精靈弓箭手，因迷路睡覺醒來後發現自己在學院，覺得精靈樹躺著舒服而留下。
    *   **語音 (CV)**：黑澤朋世
    *   **喜好/特徵**：天真無邪、嗜睡，對危險察覺度高但本人不在意。
5.  **伊麗莎·弗洛迪斯 (Elisa Flodis)**
    *   **背景**：北海武裝商船團團長之女（公主），被父親送來學習世事。
    *   **語音 (CV)**：石上靜香
    *   **喜好/特徵**：大姐姐般可靠，喜歡大海與垂釣。右眼擁有繼承自先祖的「海魔之眼」詛咒。
6.  **米埃爾·哈奈特 (Miel Harnett)**
    *   **背景**：西部沙漠地下都市出身的龍人族少女。
    *   **語音 (CV)**：高尾奏音
    *   **喜好/特徵**：開朗、話嘮、有妄想癖。食量極大（因探查龍脈耗能高）。

### 學院教職員
*   **格麗澤爾達·麥克威廉姆斯**：學院長，自稱天才法師。因打敗魔王的代價變成了小孩子身姿。
*   **蓋茨·西格特**：武技課教師，性格豪爽的戰士，外號「魔鬼」，喜歡甜食。
*   **魯納莉卡·馮頓海姆**：採集課教師，穿著高檔女僕裝，極受學生歡迎。
*   **艾瑞莎·德拉伍德伯**：奧秘課教師，精靈煉金術名手，性格嚴厲。
*   **喵姆·芙洛喵**：社會課教師，貓型獸人，同時經營小賣部。
*   **西貝爾·豪爾贊**：冒險課教師，男裝麗人（女性），重度路痴。
*   **雅爾托妮婭**：冒險者公會派遣的公務員，負責指引任務。

## 3. 形態（屬性、職業）系統說明
*   **職業分類**：目前已知的職業包括 **法師**（塞納菲娜）、**劍士**（米爾德里德）、**牧師**（吉爾伯塔）、**弓箭手**（法露露）等。
*   **元素系統**：遊戲中存在 **土、水、火、風、光、暗** 六種元素。
*   **元素牌系統**：玩家可透過「元素牌」小遊戲獲得「元素微粒」，用於提升天賦。不同角色在元素牌遊戲中擁有專屬技能（如法露露的「三重射擊」）。

## 4. 遊戲中的重要概念、歷史與術語
*   **瑪娜希斯 (Manasis)**：即精靈，是世界力量的源泉。
*   **箱庭島**：西奧德里克學院分院所在地，也是玩家與少女們生活冒險的主要舞台。
*   **瑪娜希斯技法**：利用瑪娜力量的戰鬥或生活技術。
*   **龍脈**：龍人族（如米埃爾）可以探查的能量流動。
*   **海魔之眼**：伊麗莎家族繼承的詛咒力量，需支付報酬才能借用深海之力。
*   **精靈樹**：學院內的重要地標，法露露喜歡在上面睡覺。

## 5. 相關組織
*   **西奧德里克學院 (Theodoric Academy)**：由勇者建立，旨在培育冒險者與研究瑪娜術法的核心機構。
*   **瑪娜希斯教會**：收養並撫養吉爾伯塔的組織，在各地設有大聖堂。
*   **冒險者公會**：派遣公務員（如雅爾托妮婭）至學院管理任務的組織。
*   **金匠學院 (Goldsmith Academy)**：位於北方的另一所冒險者學院，與西奧德里克學院有競爭或交流關係（如史蒂希婭即為該校學生）。
*   **阿爾扎德家族**：南方名門，史蒂希婭與格雷布所屬的家族。
