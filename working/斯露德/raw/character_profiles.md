# 《斯露德》核心角色詳細背景與性格

## 1. 法尼婭 (Fania / 法尼婭·雷霆)
*   **背景描述：** REVO 小隊的先鋒，也是導師抵達基地後見到的第一位斯露德。她擁有極高的符能天賦，擅長使用雷電元素進行戰鬥，是戰場上的突擊核心。
*   **性格特點：** 熱情開朗，充滿正能量。無論面對多麼艱難的戰況，她總能以笑容鼓勵隊友。她對正義有著執著的追求，雖然有時顯得有些莽撞，但其堅定的信念使她成為團隊中的精神支柱。

## 2. 維拉 (Vera)
*   **背景描述：** 一位經驗豐富、資歷深厚的斯露德，常在戰鬥中擔任指揮或戰術支援角色。她的過去充滿了戰鬥的痕跡，對異生體的特性與戰場環境瞭如指掌。
*   **性格特點：** 冷靜沉著，處事果斷。雖然表面上看起來有些嚴厲且不苟言笑，但實際上非常關心後輩的安危。她是導師在戰術決策上最可靠的助手，總能提供精確的分析。

## 3. 歸陌 (Guimo)
*   **背景描述：** 來自東方背景的神祕少女，其戰鬥風格融合了傳統武學與符能技術。她的身世與某個古老的家族或傳承有關，來到 REVO 似乎帶著某種特定的使命。
*   **性格特點：** 沉默寡言，自律性極強。她對自己的要求近乎苛刻，言行舉止間透著一種清冷的氣質。但在冷酷的外表下，隱藏著一顆渴望守護弱小、重塑世界秩序的心。

## 4. 空 (Sora)
*   **背景描述：** 身份極為神祕的少女，似乎與「異生體」或「遮蔽層」的核心秘密有著某種深層的聯繫。她的力量強大且不穩定，常被視為改變戰局的關鍵。
*   **性格特點：** 恬靜、空靈，有時會表現出與年齡不符的成熟與疏離感。她對世界充滿了好奇，但又帶著一種淡淡的憂傷。她與導師之間有著特殊的感應，是故事中解開世界謎團的核心人物。
