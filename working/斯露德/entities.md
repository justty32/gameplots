# 《斯露德》全景實體掃描

## 人物 (characters)
- 法尼婭 (Fania) — REVO 小隊先鋒，性格開朗的雷元素斯露德。
- 維拉 (Vera) — 經驗豐富的資深斯露德，冷靜沉著的戰術助手。
- 歸陌 (Guimo) — 來自東方、沉默自律的神祕少女，融合武學與符能。
- 空 (Sora) — 身份神祕、與異生體有深層聯繫的力量核心。
- 導師 (Mentor) — 瓦哈納艦隊指揮官，負責引領斯露德戰鬥。
- 格薇德林 (Gwendoline) — 5星角色，擁有特定戰鬥定位的斯露德。
- 椿 (Tsubaki) — 5星角色，來自東方風格背景的斯露德。
- 洛苛 (Loke) — 5星角色，具備獨特能力的斯露德少女。
- 真理 (Makoto) — 5星角色，名字帶有尋求真意意味的斯露德。
- 麥當然 (McDonnell) — 雷屬性 5星角色，擅長範圍攻擊。
- 三葉 (Mitsuba) — 5星角色，具備特定元素能力的斯露德。
- 歸陌 (Guimo) — 火屬性 5星角色，高額傷害的輸出核心。
- 耶拉 (Yela) — 風屬性 5星角色，擅長追蹤攻擊的射手。
- 白 (Shiro) — 冰屬性 5星角色，強大的控場與減速手。
- 阿萊西婭 (Alessia) — 4星角色，參與戰鬥的斯露德少女。
- 奈納 (Naina) — 4星角色，參與戰鬥的斯露德少女。
- 奧利 (Oli) — 4星角色，參與戰鬥的斯露德少女。

## 種族/陣營 (factions)
- 瓦哈納 (Vahana) — 人類反抗組織的大型移動要塞與戰艦。
- REVO — 斯露德所屬的核心戰鬥組織與小隊。
- 異生體 (Heterobiont / Heterotrees) — 遮蔽天空的神祕敵對生物。

## 事件 (events)
- 異生體入侵 — 導致人類喪失天空統治權的災難性事件。
- 天空的起點 — 導師加入 REVO 並帶領斯露德首次奪回領空的戰役。
- 遮蔽層建立 — 為了保護人類而建立的神祕屏障及其背後的真相。

## 地點 (places)
- 帆 — 人類在異生體威脅下生活的避難所。
- 瓦哈納艦橋 — 導師指揮作戰與管理日常的核心場所。
- 遺落的研究設施 — 隱藏著關於斯露德力量代價真相的廢墟。

## 物品/技術 (items)
- 式杖 (Staff / Wand) — 斯露德專用的飛行與源能轉化武裝。
- 瓦爾基里 (Valkyrie) — 斯露德使用的飛行與戰鬥武裝系統。
- 符能 (Rune Power / Source Energy) — 支撐式杖運作與斯露德戰鬥的能量。

## 概念/設定 (concepts)
- 斯露德 (Thrud) — 能與源能產生共鳴、在空中作戰的少女兵器。
- 源能 (Source Energy) — 世界觀中的核心能量來源。
- 相變反應 (Phase Change Reaction) — 不同元素斯露德交替攻擊產生的協同效應。
- 遮蔽層 — 籠罩在人類居住區上方，隔離異生體的神祕結構。
- 記憶的代價 — 獲得斯露德力量所需付出的記憶或情感代價。
