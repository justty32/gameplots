# 《Diablo》（暗黑破壞神，全系列）全景實體名單

> 跨 D1 / D2+資料片 / D3+奪魂之鐮 / Diablo Immortal / D4+仇恨之軀 與官方小說（罪惡之戰三部曲、烏雷之國、血脈、毒蛛之月等）合一。原始素材 855 頁。
> 用途：Phase 3 詞條撰寫的覆蓋 checklist。寧多勿少；重要性取捨留給 Phase 3。

## 人物 (characters)

### 至高天使（高天）
- 泰瑞爾 Tyrael — 正義/智慧天使，全系列關鍵盟友，毀世界之石、棄不朽化為凡人、重建赫拉迪姆
- 伊姆佩瑞斯 Imperius — 勇氣天使，安吉瑞議會現任領袖，驕傲魯莽，敵視人類
- 奧瑞爾 Auriel — 希望天使，議會唯一女性，外交支柱，持希望之繩
- 伊塞瑞爾 Itherael — 命運天使，讀命運卷軸
- 瑪瑟爾 Malthael — 原智慧天使、議會原領袖，世界之石消失後墮落為「死亡天使」，RoS 大反派
- 伊納留斯 Inarius — 叛逃天使，與莉莉絲共創聖修亞瑞，D4 大反派，光明大教堂創立者
- 伊蘇爾 Izual / Iz016 — 泰瑞爾副官，叛變投敵，化為墮落天使
- 維拉瑟爾 Verathiel — 困於墮落界的天使，Immortal 中被 Diablo 折磨化為「解體之歌」武器
- 哈德瑞爾 Hadriel — Immortal 後期協助流浪者的天使
- 索拉瑞恩 Solarion（伊姆佩瑞斯之矛，也算物品）／梅卡耶 Mikayel／玆別德拉 Zepydra 等次級天使
- 烏爾札爾 Urzael — 瑪瑟爾副官，化為死神，RoS Boss

### 三大魔神與次級魔神（烈焰地獄）
- 迪亞布羅 Diablo — 恐懼之王，三魔神最年幼，系列同名主反派
- 墨菲斯托 Mephisto — 仇恨之王，三魔神之長，D4 最終幕後黑手
- 巴爾 Baal — 毀滅之王，腐化世界之石
- 安達莉爾 Andariel — 痛苦少女，唯一女性魔神，D2 一幕 Boss
- 杜瑞爾 Duriel — 痛苦領主，安達莉爾「孿生」，D2 二幕 Boss
- 貝利亞 Belial — 謊言之王，D3 二幕 Boss
- 阿茲莫丹 Azmodan — 罪惡之王，D3 三幕 Boss
- 塔哈麥特 Tathamet — 七頭巨龍，原初之惡，所有魔神之源
- 莉莉絲 Lilith — 墨菲斯托之女，仇恨之女，聖修亞瑞與奈非天之母，D4 主反派
- 路西恩 Lucion — 莉莉絲之兄，三尊神 Triune 幕後
- 史卡恩 Skarn — Diablo 副官，Immortal 主反派，建墮落界
- 屠夫 The Butcher — D1 著名魔物
- 西妲雅 Cydaea／戈姆 Ghom — 阿茲莫丹麾下罪惡副官
- 瑪格達 Maghda — 信徒會領袖，殺死凱恩
