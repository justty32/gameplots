# 深空之眼 實體名單 (entities.md)

## 人物 (characters)
- 管理員 (Administrator) — 玩家扮演，深空之眼的戰術指揮官。
- 奧丁 (Odin) — 深空之眼組織的建立者，原初修正者。
- 薇兒丹蒂 (Verthandi) — 特別作戰科成員，性格開朗的正義少女。
- 波塞冬 (Poseidon) — 第九部門成員，溫柔的支援型修正者。
- 休 (Hera) — 第九部門副隊長，深空之眼的實際管理者。
- 月讀 (Tsukuyomi) — 雉小隊隊長，冷靜強大的劍客。
- 庚辰 (Gengchen) — 虛恒四方院的建立者。
- 陵光 (Lingguang) — 虛恒四方院代理人之一。
- 孟章 (Mengzhang) — 虛恒四方院代理人之一。
- 執明 (Zhiming) — 虛恒四方院代理人之一。
- 金烏 (Jinwu) — 羲和的弟子。
- 哈迪斯 (Hades) — 歐莫菲斯星虹議會首席，繆斯協會主理人。
- 雅典娜 (Athena) — 鈴蘭騎士團團長。
- 俄刻阿諾斯 (Oceanus) — 星虹議會第三席。
- 阿努比斯 (Anubis) — 科爾蓋重大犯罪對策部成員。
- 芭絲特 (Bastet) — 科爾蓋重大犯罪對策部成員。
- 海拉 (Hel) — 送葬者小隊負責人。
- 阿修羅 (Asura) — 夏什瓦特區域的強大戰鬥修正者。
- 前鬼坊天狗 (Zenki-bo Tengu) — 雷小隊成員。
- 托爾 (Thor) — 雷小隊成員。
- 阿波羅 (Apollo) — 雷小隊成員。

## 種族/陣營 (factions)
- 深空之眼 (Aether Gazer) — 由奧丁建立的對抗視骸的修正者組織。
- 真櫻 (Shinonome) — 以和風神話為背景的修正者神系。
- 聖樹 (Yggdrasill) — 以北歐神話為背景的修正者神系。
- 奧林匹斯 (Olympus) — 以希臘神話為背景的修正者神系。
- 尼羅 (Nile) — 以埃及神話為背景的修正者神系。
- 眾星 (Asterism) — 以印度/其他神話為背景的修正者神系。
- 天垣 (Tianyuan) — 以華夏神話為背景的修正者神系。
- 第九部門 — 深空之眼下屬的核心管理與作戰部門。
- 特別作戰科 — 處理緊急危機的精銳部門。
- 雉小隊 — 月讀帶領的作戰小隊。
- 雷小隊 — 托爾、阿波羅等組成的作戰小隊。
- 四方院 — 虛恒地區的核心管理組織。
- 天祿貿易 — 虛恒地區的商業組織。
- 星虹議會 — 歐莫菲斯地區的統治機構。
- 鈴蘭騎士團 — 歐莫菲斯的守護武裝。
- 繆斯協會 — 歐莫菲斯的藝術文化組織。
- 彌樓衍 — 夏什瓦特地區的組織。
- 阿卡德公會 — 夏什瓦特地區的組織。
- 奧林匹亞速運 — 艾因索菲地區的物流組織。
- 迷鐵之家 — 諾伊漢薩地區的組織。

## 事件 (events)
- 舊世界毀滅 — 人類文明遭遇災難，被迫轉移至虛擬世界的起始事件。
- 視骸危機 — 視骸侵蝕蓋亞數據源層引發的持續性生存威脅。

## 地點 (places)
- 蓋亞 (Gaea) — 舊世界毀滅後，人類意識居住的超級電腦虛擬世界。
- 原質區 (Sephirah Zone) — 蓋亞世界中的地理與文化行政劃分。
- 表層 (Surface) — 人類意識體正常生活的虛擬空間。
- 源層 (Source Layer) — 儲存原始數據、具備修復功能的底層空間。
- 虛恒 (Xu Heng) — 東方風格的原質區。
- 歐莫菲斯 (Omorfies) — 藝術與美學之都原質區。
- 笹波 (Sasanami) — 和風文化原質區。
- 夏什瓦特 (Shashwat) — 具有特定文化背景的原質區。
- 科爾蓋 (Quorgie) — 設有重大犯罪對策部的原質區。
- 艾因索菲 (Ain Soph) — 商業發達的原質區。
- 諾伊漢薩 (Neuhansa) — 工業發達的原質區。

## 物品/技術 (items)
- 權鑰 (Access Key) — 修正者使用的武器，系統權限的具象化。

## 概念/設定 (concepts)
- 修正者 (Modifier) — 蓋亞系統為了應對危機而進化的神名化實體。
- 視骸 (Visbane) — 誕生於源層、意圖毀滅人類文明的數據怪物。
- 蓋亞系統 (Gaea.Zero) — 維持虛擬世界運行的核心電腦系統。
- 神位 — 衡量修正者能力階級與強度的指標。
