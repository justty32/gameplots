# 終焉誓約 原始資料概覽

## 一、 世界觀背景
*   **創世傳說**：創世之靈「科洛斯」以無比智慧創造了世間萬物，並花費千年時間挑選數名女武神守護這個世界。
*   **黑暗時代**：魔神「希摩爾」曾率領魔族肆虐「艾恩大陸」長達五百年。在那段黑暗時代，眾神紛紛隕落，文明幾乎覆滅。
*   **英雄封印**：人族青年「羅蘭」引領各族發起反抗，最終以生命為代價，手持失落神具將魔神封印，為世界找回了和平與光明。
*   **現狀**：和平持續了數十年後，一股不具名的邪惡力量悄然侵入，原本平靜的生活再度崩塌。

## 二、 核心劇情大綱
故事圍繞著少年「西蒙」展開，他為了追逐兄長的背影，踏上了尋覓真相的命運旅程。在冒險過程中，玩家將與西蒙一同經歷以下核心事件：
1.  **粉碎野心**：揭發並阻止右相覬覦王座的陰謀，尋回遺落民間的皇族之女。
2.  **解除詛咒**：打破永夜魔法的詛咒，協助精靈公主對抗邪惡的「誓魔者」。
3.  **守護家園**：見證年幼的天空城女王挺身而出，守護自己的家園。
4.  **終焉抉擇**：在「終焉之刻」降臨時，少年必須舉起利劍，運用智慧與勇氣揭開隱藏在真相背後的終極真相，為守護當初締結的誓約而戰。

## 三、 主要角色列表
*   **西蒙·奧茲 (Simon Oz)**：本作主角，性格樂觀莽撞，對劍技有強烈執著，目標是成為「帝國第一劍士」。
*   **溫麗·芙蕾雅 (Wenli Freya)**：擁有強大魔法能力的少女，自尊心極強且愛好美食，曾多次獲得大胃王冠軍。
*   **納塔娜·拉比納 (Natana Labina)**：「魔耳朵」馬戲團團長，擅長將魔法融入魔術表演中，帶給觀眾歡笑。
*   **艾緹歐·卡蘭登 (Aiteo Calanden)**：外表平易近人，實則隱藏著不為人知的黑暗與狂氣，只信任手中的劍與蘭奇。
*   **洛伊·歐斯 (Loyi Os)**：歐斯家族的大小姐，視溫麗為勁敵。雖然表現出「中二病」的姿態，但那是她保護自己的方式。
*   **阿德琳·貝爾 (Adeline Bell)**：出身德魯伊部落，能變換為貓熊形態。外表如高嶺之花，內心卻對英俊男子有豐富的幻想。
*   **葛萊蒂絲 (Gladys)**：高貴的精靈族公主，天真善良且熱愛自然，但因種族矛盾對人類抱有戒備。
*   **米蘭達·伊頓·格特魯格 (Miranda)**：納雅軍隊總司令，具備極強的作戰與指揮能力，身負鋼鐵翼刃作戰。
*   **稻荷葛葉 (Inari Kuzuha)**：三尺境守護者之一，守衛星穹殿。雖然表面健談，實際上是個有社交恐懼症的宅狐。
*   **蘭奇·奧茲 (Lanchi Oz)**：西蒙的兄長，是西蒙踏上旅程的重要契機。
