# 《未來戰》(Counter:Side) 全景實體掃描

## 人物 (characters)
- 管理者 (Administrator / CEO) — 玩家扮演的角色，科芬公司的 CEO，經歷無數次輪迴的永生者。
- 希爾德 (Hilde) — 芬里爾小隊導師，被稱為「師父」，擁有極強戰鬥力與長壽的傳奇制裁者。
- 劉米娜 (Yoo Mina) — 本作主角，天賦異稟的新人制裁者，繼承了神祕的「龍血」。
- 周時雲 (Joo Shiyoon) — 芬里爾小隊成員，平時玩世不恭，繼承了家族神祕的「龍」之力。
- 徐倫 (Seo Yoon) — ALT小隊隊長，原為實驗體，擁有出色的戰術頭腦與生存本能。
- 羅瑜斌 (Na Yubin) — 「六翼」組織領導者，曾是希爾德的弟子，為達成目標不擇手段。
- 李智秀 (Lee Jisoo) — 「六翼」成員，實力強大的制裁者，負責執行羅瑜斌的計畫。
- 曉琳 (Xiao Lin) — ALT小隊狙擊手，性格冷靜，是極受玩家歡迎的強力遠程輸出。
- 金素蘋 (Kim Sobin) — ALT小隊重火力手，對武器有著異常執著。
- 劉珍 (Yoo Jin) — ALT小隊前衛，性格開朗，負責小隊的氛圍。
- 克隆皇后 (Replacer Queen) — 克隆集團核心成員，擁有操控強大侵蝕能量的能力。
- 克隆主教 (Replacer Bishop) — 克隆集團核心成員，負責集團的技術與策略。
- 申娜拉 (Shin Nara / 大主教) — 劉米娜的昔日同學，因侵蝕災難而黑化成為救贖騎士團大主教。
- 尤娜 (Yuna Springfield) — 斯特雷加咖啡廳的魔女，擁有強大的魔法能力。
- 七原千冬 (Chifuyu Nanahara) — 七原家族武士，性格剛毅，守護家族傳統。
- 七原千夏 (Chinatsu Nanahara) — 七原家族成員，千冬的妹妹，擁有治癒能力。
- 伊莉莎白 (Elizabeth Pendragon) — 普利德溫機關首領，高貴的吸血鬼族裔。
- 羅莎莉亞 (Rosaria le Friede) — 普利德溫機關成員，擁有極高輸出的魔女。
- 赫里遜 (Horizon) — 赫里遜金融的冷酷管理者，實為強大的機械體。
- 維羅妮卡 (Veronica) — 弗羅拉女僕團團長，擁有不死之身的強大戰士。

## 種族/陣營 (factions)
- 科芬公司 (Coffin Company) — 玩家經營的民間軍事企業，主要應對侵蝕威脅。
- 管理局 (Administration) — 維護世界秩序的公的機關，負責制裁者的管理與登記。
- 舊管理局 (Old Administration) — 過去曾經輝煌的全球管理組織，在「管理失敗」後瓦解。
- 芬里爾小隊 (Fenrir Squad) — 由希爾德帶領的主角小隊，科芬公司的王牌。
- ALT小隊 (ALT Squad) — 由徐倫帶領的雇傭兵小隊，成員多為實驗體背景。
- 六翼 (Six Wings) — 由羅瑜斌領導的神祕組織，在暗中推動世界的變局。
- 克隆集團 (Replacer Syndicate) — 試圖將人類轉化為「新人類」以應對侵蝕的極端組織。
- 七原家族 (Nanahara Family Alliance) — 守護東方傳統與禁地的強大家族聯盟。
- 普利德溫機關 (Prydwen Institute) — 歷史悠久的對侵蝕體專門機關。
- 救贖騎士團 (Savior Knights) — 崇尚救贖、帶有宗教色彩的武裝組織，大主教為其首領。
- 斯特雷加 (Strega) — 由魔女組成的秘密組織，平時經營咖啡廳。
- 弗羅拉女僕團 (Flora Maid Service) — 表面提供管家服務，實為戰鬥力強悍的武裝小隊。

## 事件 (events)
- 管理失敗 (Administration Failure) — 過去人類對抗侵蝕失敗，導致文明幾乎崩壞的慘劇。
- 泰拉賽德事件 (Terraside Incident) — 主線第一季高潮，克隆集團引發的大規模侵蝕危機。
- 克里福特遊戲 (Qliphoth Game) — 魔王們為了爭奪現實世界而發動的周期性侵蝕毀滅儀式。
- 20年前的毀滅性侵蝕戰爭 — 導致舊管理局瓦解、多位傳奇人物命運改變的關鍵戰役。
- 北極攻防戰 (Arctic Battle) — 揭示了世界觀深層真相與「女皇」起源的重要事件。

## 地點 (places)
- 現實世界 (Normal Side) — 人類正常居住的世界，正不斷受到異界侵蝕。
- 異界 (Counter Side) — 與現實並行的異空間，充滿侵蝕能量與怪物。
- 花天 (Huatian) — 具有強烈東方色彩的都市，由白西與黑東兩大勢力統治。
- 管理局制裁者學院 (Administration Counter Academy) — 專門培養與監控制裁者的官方機構。

## 物品/技術 (items)
- 制裁者手環 (Counter Watch) — 制裁者用來穩定並控制異界能量的關鍵設備。
- 永恆之金 (Eternium) — 來自異界的強大動力礦石，也是各方勢力爭奪的核心資源。
- 次元艦 (Ship) — 具備空間跳躍能力，能進入侵蝕區域執行任務的高科技船艦。
- 克隆因子 (Replacer Factor) — 能讓人體強化但會導致嚴重侵蝕後遺症的人工基因技術。

## 概念/設定 (concepts)
- 制裁者 (Counters) — 覺醒了特殊能力、能運用異界能量對抗侵蝕體的人類。
- 侵蝕體 (Corrupted Objects) — 來自異界的異質怪物，會導致周遭物質發生侵蝕崩壞。
- 異界效應 (CSR - Counter Side Effect) — 現實世界受到異界能量干擾而產生的異常現象。
- 現實干預能力 (CRF - Counter Reality Force) — 改變現實物理定律的力量，力量愈強則干預程度愈大。
- 逆卡巴拉因子 / 龍血 — 極為強大的異界能量源，繼承者擁有弒神般的潛力。
- 影 (Shadow) — 存在於異界中，現實世界住人的扭曲投影。
- 魔王 (Demon Lord) — 異界中具備支配地位的強大存在，克里福特遊戲的主角。
