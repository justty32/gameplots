# 碧藍幻想 (Granblue Fantasy) 全景實體掃描

## 人物 (characters)
- 格蘭 / 姬塔 (Gran / Djeeta) — 本作主角，被稱為「特異點」，與露莉亞生命連結。
- 露莉亞 (Lyria) — 蒼之少女，能與星晶獸溝通並吸收其力量。
- 碧 (Vyrn) — 主角的搭檔，愛吃蘋果的小紅龍，真實身分與創世神有關。
- 卡塔莉娜 (Katalina Alize) — 原帝國騎士，為了守護露莉亞而背叛帝國。
- 拉卡姆 (Rackam) — 天才操舵手，負責駕駛格蘭賽法號。
- 伊歐 (Io Euclase) — 天才魔法少女，為了尋找師父而加入騎空團。
- 尤金 (Eugen) — 資深傭兵，與艾爾斯特帝國有著深厚淵源。
- 蘿賽塔 (Rosetta) — 充滿謎團的美女，真實身分為星晶獸「蘿潔薇雅」。
- 芙莉西亞 (Freesia von Bismarck) — 艾爾斯特帝國總理，早期劇情的幕後黑手。
- 歐契絲 (Orchis) — 艾爾斯特帝國的少女，後成為帝國的女王。
- 阿波羅尼亞 / 黑騎士 (Apollonia Vaar) — 七曜騎士之一，管理法塔格蘭德空域。
- 希耶提 (Siete) — 十天眾的領導者，擁有極其強大的劍術與魔力。
- 聖德芬 (Sandalphon) — 原天司長代理，在經歷動亂後繼承了天司長之位。
- 路西法 (Lucilius) — 星之民的研究者，企圖毀滅世界的最終反派之一。
- 別西卜 (Beelzebub) — 星之民的叛徒，追求力量的極致。
- 貝利亞 (Belial) — 墮天司，路西法的狂熱信徒，混亂的推動者。
- 卡莉歐斯托羅 (Cagliostro) — 鍊金術的開創者，以變換身體來獲得長生。

## 種族/陣營 (factions)
- 星之民 (Astrals) — 數百年前從異世界侵略而來的強大種族，創造了星晶獸。
- 空之民 (Skydwellers) — 空之世界的原住民，包含人類、艾倫、多拉夫、哈文。
- 艾爾斯特帝國 (Erste Empire) — 早期企圖利用星晶獸力量統一空域的軍事帝國。
- 七曜騎士 (Seven Luminary Knights) — 直屬於真王的七位最強騎士，維持空域秩序。
- 十天眾 (Eternals) — 空之世界最強的十位武裝高手組成的集團。
- 十賢者 (Evokers) — 與塔羅牌系列星晶獸契約的極端強大個體。
- 組織 (The Society) — 專門獵殺星晶獸並對抗「非」的秘密武裝組織。
- 非 (The Foe) — 與「組織」對立的神秘組織，涉及多項禁忌計畫。
- 月之民 (Moondwellers) — 居住在月球上的古代人類後裔，擁有高度科技。
- 星之民叛軍 (Fallen Angels) — 由路西法領導，反抗星之民體系的天司。

## 事件 (events)
- 霸空戰爭 (War of the Sky) — 五百年前空之民反抗星之民統治的大規模戰爭。
- 墮天司之亂 (Fallen Angel Rebellion) — 路西法與別西卜發動的反叛，重塑了天司體系。
- 天為何而藍 (What Makes the Sky Blue) — 圍繞聖德芬與路西法復活計畫的世界性危機。
- 月之慶典 (Stay Moon) — 月之民對空之世界發動的大規模技術侵略戰爭。

## 地點 (places)
- 法塔格蘭德空域 (Phantagrande Skydom) — 故事起點所在的空域，受格林風盆圍繞。
- 傑加格蘭迪空域 (Zegagrande Skydom) — 位於西方，Relink 故事的主要舞台。
- 納爾哈格蘭德空域 (Nalhegrande Skydom) — 第二部主線舞台，以高度科技聞名。
- 辛肯斯提爾 (Zinkenstill) — 主長的故鄉，一個安靜的小島。
- 阿加斯提亞 (Agastia) — 艾爾斯特帝國的首都。
- 伊斯塔魯西亞 (Estalucia) — 傳說中的星之島，位於天空的盡頭。
- 幽世 / 赤之水平線 (The Otherworld) — 空之世界下方的混沌與死亡之地。

## 物品/技術 (items)
- 格蘭賽法號 (Grandcypher) — 騎空團的母船，由古代技術製造的傳奇空艇。
- 露莉亞的筆記 (Grimoire) — 記載著露莉亞吸收的星晶獸力量與旅程點滴。
- 七曜之武 (Luminary Weapons) — 七曜騎士所持有的強大神器。
- 機神 (Automagods) — 月之民所使用的超高度科技自動兵器。

## 概念/設定 (concepts)
- 星晶獸 (Primal Beasts) — 星之民製造的半神兵器，擁有各種管理世界的力量。
- 生命連結 (Life Link) — 主角與露莉亞共感生命的契約，使兩人命運一體。
- 特異點 (The Singularity) — 能擺脫命運支配、改變世界走向的特殊個體（主角）。
- 混沌物質 (Chaos Matter) — 幽世的能量，對星之民與空之民皆有劇毒。
- 創世神巴哈姆特 (Bahamut) — 世界的創造者與毀滅者，被分割為空之神與星之神。
