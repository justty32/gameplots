# 東京戰記 (東京喰種) 全景實體掃描

## 人物 (characters)
- 金木研 (Kaneki Ken) — 本作主角，從人類轉變為半喰種。
- 霧嶋董香 (Kirishima Touka) — 女主角，安定區咖啡廳服務生，代號「兔子」。
- 佐佐木琲世 (Haise Sasaki) — 金木研失憶後的身份，Qs班指導者。
- 永近英良 (Nagachika Hideyoshi) — 金木研的摯友，外號「Hide」。
- 神代利世 (Kamishiro Rize) — 引發一切事件的關鍵，代號「大食客」。
- 芳村功善 (Yoshimura Kuzen) — 咖啡廳店長，代號「不殺之梟」。
- 四方蓮示 (Yomo Renji) — 店長的得力助手，代號「烏鴉」。
- 笛口雛實 (Fueguchi Hinami) — 被金木照顧的幼年喰種。
- 西尾錦 (Nishio Nishiki) — 藥學系大學生，代號「大蛇」。
- 入見佳耶 (Irimi Kaya) — 咖啡廳店員，原「黑色多伯曼」首領。
- 古間圓兒 (Enji Koma) — 咖啡廳店員，原「魔猿」首領。
- 亞門鋼太朗 (Amon Koutarou) — 正義感極強的 CCG 搜查官。
- 真戶吳緒 (Mado Kureo) — 亞門的搭檔，對昆克異常執著。
- 真戶曉 (Mado Akira) — 真戶吳緒的女兒，優秀的女性搜查官。
- 鈴屋什造 (Suzuya Juuzou) — 戰鬥天賦極高的特等搜查官。
- 有馬貴將 (Arima Kishou) — CCG最強搜查官，被稱為「白色死神」。
- 篠原幸紀 (Yukinori Shinohara) — 什造的導師，資深特等。
- 瀧澤政道 (Seidou Takizawa) — 搜查官，後轉變為半喰種「貓頭鷹」。
- 艾特 / 高槻泉 (Eto / Takatsuki Sen) — 青銅樹首領，獨眼之梟。
- 多多良 (Tatara) — 青銅樹幹部，來自中國的喰種。
- 野呂 (Noro) — 擁有驚人再生能力的青銅樹幹部。
- 霧嶋絢都 (Kirishima Ayato) — 董香的弟弟，代號「黑兔」。
- 壁虎 / 大守八雲 (Yamori) — 13區殘暴喰種，對金木施以拷問。
- 唄 (Uta) — 面具師，代號「無臉」。
- 系璃 (Itori) — 情報販子，與四方、唄是舊識。
- 帆糸蘿瑪 (Roma Hoito) — 小丑成員，創始者。
- 月山習 (Tsukiyama Shuu) — 美食家，對金木有病態執著。
- 萬丈數一 (Kazuichi Banjou) — 追隨金木的溫厚喰種。
- 嘉納明博 (Kanou Akihiro) — 將金木改造的瘋狂醫生。
- 瓜江久生 (Kuki Urie) — Qs班班長，極具野心。
- 不知吟士 (Ginshi Shirazu) — Qs班成員，性格豪爽。
- 六月透 (Tooru Mutsuki) — Qs班成員，有複雜過去。
- 米林才子 (Saiko Yonebayashi) — Qs班成員，高天賦宅女。

## 種族/陣營 (factions)
- 喰種 (Ghoul) — 外表如常人，以人肉為食的亞人種。
- 半喰種 (Half-Ghoul) — 人類與喰種的混血或改造體。
- 安定區 (Antei-ku) — 20區的咖啡廳，協助弱小喰種。
- CCG (喰種對策局) — 專門應對喰種的國家機關。
- 青銅樹 (Aogiri Tree) — 強大且激進的喰種組織。
- 小丑 (Clowns) — 身份神秘、以混亂為樂的組織。
- V — 隱藏在世界背後、維持平衡的組織。
- 昆克斯班 (Quinx Squad / Qs班) — 體內植入赫包的 CCG 特殊部隊。
- 黑色多伯曼 (Black Dog) — 入見佳耶曾領導的武鬥派組織。
- 魔猿 (Devil Ape) — 古間圓兒曾領導的組織。

## 事件 (events)
- 嘉納醫院慘劇 — 金木研被植入利世器官的起點。
- 11區青銅樹圍剿戰 — CCG 對抗青銅樹的大型戰役。
- 安定區(20區)討伐戰 — CCG 殲滅不殺之梟的戰役。
- 流島登陸戰 — 對抗青銅樹殘餘勢力的戰役。
- 奎庫利亞收容所攻防戰 — 喰種監獄的大規模劫獄與防守。
- 龍之戰 — 作品末期影響全東京的災難性事件。

## 地點 (places)
- 20區 — 故事起點，安定區所在地。
- 安定區咖啡廳 (Antique) — 喰種與人類共存的避風港。
- 奎庫利亞 (Cochlea) — CCG 的喰種收容所。
- 海萊 (HySy ArtMask Studio) — 唄經營的面具店。
- CCG 總部 — 位於1區，喰種對策局的核心。
- 魯斯島 (流島) — 青銅樹的基地。

## 物品/技術 (items)
- 赫包 — 儲存Rc細胞的器官。
- 庫因克 (Quinque) — 以赫子製成的反喰種武器。
- 昆克斯手術 — 在搜查官體內植入赫包的技術。
- Rc檢查門 — 偵測高Rc濃度的感應器。
- CRc氣體 — 抑制Rc細胞活性的特殊氣體。

## 概念/設定 (concepts)
- Rc細胞 — 喰種與人類體內皆有的特殊細胞。
- 赫子 (Kagune) — 喰種的捕食器官。
- 羽赫 (Ukaku) — 肩部附近，輕快且具射程。
- 甲赫 (Koukaku) — 肩胛骨下，硬度高適合防禦。
- 鱗赫 (Rinkaku) — 腰部附近，強大的打擊力與再生力。
- 尾赫 (Bikaku) — 尾椎附近，平衡感極佳。
- 赫者 — 吃掉同類的喰種產生的變異形態。
- Rc值 — 體內Rc細胞的濃度數值。
- 白日庭 — 培養混血兒與半人類的祕密機構。
