# 《棕色塵埃2 (Brown Dust 2)》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物
- [拉德爾 (Lathel)](characters.md#拉德爾-lathel) — 盧戈鎮採藥少年，血刻印體質，本作第一主角。
- [悠絲緹亞 (Justia)](characters.md#悠絲緹亞-justia) — 綽號「皎潔死神」，卡里安帝國騎士。
- [莎赫拉查德 (Scheherazade)](characters.md#莎赫拉查德-scheherazade) — 綽號「蒼藍魔女」，魔法專家。
- [格雷 (Gray)](characters.md#格雷-gray) — 綽號「迷霧神射手」，冷靜的傭兵。
- [威格 (Wiggle)](characters.md#威格-wiggle) — 不死族炸彈兵。
- [泰瑞絲 (Teresse)](characters.md#泰瑞絲-teresse) — 莎赫拉查德的姐姐。
- [米莎 (Misha)](characters.md#米莎-misha) — 拉德爾的妹妹，序章犧牲。

## 種族 / 陣營
- [科庫托斯 (Cocytus)](factions.md#科庫托斯-cocytus) — 貫穿全劇的黑魔法組織。
- [卡里安帝國 (Carian Empire)](factions.md#卡里安帝國-carian-empire) — 悠絲緹亞所屬的國家。
- [不死族 (Undead)](factions.md#不死族-undead) — 受到阿德林影響或黑魔法復活的生物。

## 事件
- [血刻印覺醒](events.md#血刻印覺醒) — 拉德爾釋放泰拉力量的轉折點。
- [虛無之星 (Empty Star)](events.md#虛無之星-empty-star) — 故事的序章，講述了英雄的悲劇誕生。
- [火與冰 (Fire and Ice)](events.md#火與冰-fire-and-ice) — 主角團在邊境地區的首次重大任務。
- [鮮血之宴 (Blood Banquet)](events.md#鮮血之宴-blood-banquet) — 第一部劇情的巔峰與重大悲劇轉折。

## 地點
- [斯貝恩大陸 (Svern Continent)](places.md#斯貝恩大陸-svern-continent) — 故事發生的主舞台。
- [盧戈鎮 (Lugo)](places.md#盧戈鎮-lugo) — 拉德爾成長的故鄉。
- [惡魔城](places.md#惡魔城) — 充滿挑戰的神祕塔類建築。

## 概念 / 設定
- [血刻印](concepts.md#血刻印) — 植入人體內的黑暗魔力核心。
- [皎潔死神](concepts.md#皎潔死神) — 獵殺黑魔法師的特殊身份。
- [卡帶 (Game Cartridges)](concepts.md#卡帶-game-cartridges) — 連結不同時空與劇情的媒介。
- [阿德林 (Adeline)](concepts.md#阿德林-adeline) — 強化生物但奪取理性的神祕藥劑。
