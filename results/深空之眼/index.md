# 《深空之眼》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)。

## 人物
- [管理員](characters.md#管理員) — 深空之眼組織的戰術指揮官，玩家扮演。
- [奧丁](characters.md#奧丁) — 深空之眼組織的建立者，原初修正者。
- [薇兒丹蒂](characters.md#薇兒丹蒂) — 深空之眼特別作戰科成員，故事初期的核心夥伴。
- [波塞冬](characters.md#波塞冬) — 第九部門成員，溫柔的支援型修正者。
- [休](characters.md#休) — 第九部門副隊長，深空之眼的實際管理者。
- [月讀](characters.md#月讀) — 雉小隊隊長，笹波地區的高人氣劍客。
- [庚辰](characters.md#庚辰) — 虛恒四方院建立者，天垣神系領袖人物。
- [陵光 / 孟章 / 執明](characters.md#陵光--孟章--執明) — 虛恒四方院的代理人，守護虛恒的不同方位。
- [哈迪斯](characters.md#哈迪斯) — 歐莫菲斯星虹議會首席，繆斯協會主理人。
- [雅典娜](characters.md#雅典娜) — 鈴蘭騎士團團長，歐莫菲斯的守護者。
- [阿修羅](characters.md#阿修羅) — 夏什瓦特區域的強大戰力。
- [海拉](characters.md#海拉) — 送葬者小隊負責人。
- [阿努比斯 / 芭絲特](characters.md#阿努比斯--芭絲特) — 科爾蓋重大犯罪對策部成員。

## 種族 / 陣營
- [深空之眼](factions.md#深空之眼) — 由奧丁建立的對抗視骸的修正者組織，對外是一家大型綜合投資公司。
- [真櫻](factions.md#真櫻) — 以和風神話為背景的修正者神系。
- [聖樹](factions.md#聖樹) — 以北歐神話為背景的修正者神系。
- [奧林匹斯](factions.md#奧林匹斯) — 以希臘神話為背景的修正者神系。
- [天垣](factions.md#天垣) — 以華夏神話為背景的修正者神系。
- [四方院](factions.md#四方院) — 虛恒地區的核心管理與文化組織。
- [星虹議會](factions.md#星虹議會) — 歐莫菲斯地區的統治機構。

## 地點
- [蓋亞](places.md#蓋亞) — 舊世界毀滅後，人類意識上傳後居住的超級電腦虛擬世界。
- [源層](places.md#源層) — 蓋亞的數據底層空間。
- [虛恒](places.md#虛恒) — 具有東方文化風格的原質區。
- [歐莫菲斯](places.md#歐莫菲斯) — 藝術與美學之都原質區。
- [笹波](places.md#笹波) — 和風文化原質區。
- [諾伊漢薩](places.md#諾伊漢薩) — 重工業與科技發達的原質區。

## 物品 / 技術
- [權鑰](items.md#權鑰) — 修正者所使用的特殊武器。
- [神位](items.md#神位) — 衡量修正者力量等級與權限的指標。

## 概念 / 設定
- [修正者](concepts.md#修正者) — 蓋亞系統為了應對數據危機而進化出的「人格化實體」。
- [視骸](concepts.md#視骸) — 誕生於數據源層的怪物，意圖毀滅人類文明。
- [蓋亞系統](concepts.md#蓋亞系統) — 維持虛擬世界運行的超級電腦核心系統。
- [原質區](concepts.md#原質區) — 蓋亞世界中的地理與行政劃分單位。
