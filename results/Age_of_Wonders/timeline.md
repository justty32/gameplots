# 《Age of Wonders》經典三部曲 — 編年表

> 線性時間骨架。細節見 [events.md](events.md)；速覽見 [synopsis.md](synopsis.md)。
> 紀年：**LIR**=因尼歐克治世、**QJR**=茱莉亞女王治世（QJR 1 ≈ AoW1 終局後）、**YC**=聯邦年（YC 1 = QJR 395）。

## 第一紀：因尼歐克的青翠王國（AoW1 的時代）

> 不死的精靈巫王統治近千年；人類到來、舊王被弒、各族混戰，最終以奇蹟谷之戰收場。

- [神話時代] — **盟約** — 高人與初代精靈立約：精靈開拓世界後須讓位人類 → [詳見](events.md#盟約the-covenant)
- 0 LIR — **因尼歐克加冕** — 精靈巫王登基，父退隱永恆界
- 800–801 LIR — **高人來訪** — 高人預告人類將至、要求精靈退居永恆界；因尼歐克拒絕並隱瞞，另創守護者
- 878 LIR — **米安多誕生** — 元配艾蓮諾難產而死，因尼歐克再娶艾爾溫，米安多生恨
- 962–965 LIR — **人類抵達** — 人類經血島衝突抵奇蹟谷，守護者准其定居
- 965–966 LIR — **首次衝突** — 黑暗精靈煽動下各族械鬥 → [詳見](events.md#首次衝突the-first-conflict)
- 969 LIR — **茱莉亞誕生** — 被宣為亡妻「替代」，米安多更怒
- 970 LIR — ★ **精靈宮廷之陷落** — 人類起義弒因尼歐克，米安多暗中背叛父親 → [詳見](events.md#精靈宮廷之陷落fall-of-the-elven-court)
- 974 LIR — **席爾凡努斯立邦** — 獨立精靈邦成立
- 977 LIR — **米安多被擁立** — 黑暗精靈擁其為王
- 985 LIR — **風暴邪教成立** — 米安多創教誓滅人類 → [詳見](events.md#精靈宮廷之陷落fall-of-the-elven-court)
- 999 LIR — **艾爾多島之取得** — 漢·賓格以一吻換島予流亡精靈 → [詳見](events.md#艾爾多島之取得the-acquisition-of-aldor)
- 1000 LIR — **混亂之治** — 諸盟瓦解、米安多爭位 → [詳見](events.md#混亂之治reign-of-chaos)
- 1025 LIR — **茱莉亞入守護者** — 史上最年輕的成員
- 1082–1098 LIR — **阿茲拉克入侵** — 沙漠帝國北侵，被守護者同盟擊退 → [詳見](events.md#阿茲拉克入侵azrac-invasion)
- 1101 LIR — **吹響亡者之角** — 梅蕾妮絲釋放不死大軍 → [詳見](events.md#吹響亡者之角blowing-of-the-trump)
- 1102–1149 LIR — **不死族戰爭與高人介入** — 不死橫掃，1146 高人降臨擊退 → [詳見](events.md#不死族戰爭與高人介入undead-wars)
- 1164 LIR — **邪教納死靈術** — 邪教與不死簽約，死靈術士滲入
- 1204 LIR — **茱莉亞升任領袖** — 擊敗獸人後領導守護者
- 1206–1216 LIR — **沉默十年** — 詭異平靜下各族瘋狂擴軍 → [詳見](events.md#沉默十年decade-of-silence)
- 1216 LIR — ★ **奇蹟谷之星出現** — 高人所置天兆，沉默告終 → [詳見](events.md#奇蹟谷之星出現appearance-of-the-star)
- 1216 LIR — **艾爾溫遇刺** — 米安多派哥布林暗殺 → [詳見](events.md#艾爾溫遇刺assassination-of-elwyn)
- 1216 LIR — **深密爾陷落** — 梅蕾妮絲崩毀矮人大城 → [詳見](events.md#深密爾陷落destruction-of-deepmir)
- 1216 LIR — **因尼歐克復活** — 於「骷髏」復活，卻反噬邪教 → [詳見](events.md#因尼歐克復活resurrection-of-inioch)
- 1216 LIR — ★ **奇蹟谷之戰** — 五方混戰，茱莉亞全勝，米安多戰死 → [詳見](events/battle-for-the-valley-of-wonders.md)

## 第二紀：茱莉亞女王與巫王寶座（AoW2 + Shadow Magic 的時代）

> 茱莉亞稱后、奇蹟谷重歸精靈；人類被逐後四處侵擴，巫師圈反目掀起浩劫，終致暗影惡魔降臨。

- QJR 元年 — **第二紀開始** — 茱莉亞登基，將永恆界交予加百列領導巫師圈
- 第二紀初 — **人類被逐與擴張** — 人類流離他鄉、侵入他族之地 → [詳見](events.md#人類被逐與第二紀擴張displacement-of-humans)
- 第二紀 — **阿茲拉克帝國覆滅** — 人類滅之，亞卡棄之轉造虎人 → [詳見](events.md#阿茲拉克帝國覆滅fall-of-the-azrac-empire)
- 約 QJR 350 — **巫師圈背叛／巫師浩劫** — 巫師反目，跨界災難，因尼歐克之靈誘叛徒 → [詳見](events.md#巫師圈背叛與巫師浩劫betrayal-of-the-circle--wizards-cataclysms)
- 約 QJR 350 — ★ **梅林集六球之旅** — 梅林獲加百列所救、依序精通六球 → [詳見](events.md#梅林集六球之旅merlins-quest)
- QJR 350 末 — ★ **因尼歐克附身茱莉亞、梅林登基** — AoW2 終局，梅林逐出因尼歐克、登上巫王寶座 → [詳見](events.md#因尼歐克附身茱莉亞與梅林登基possession--ascension)
- 緊接其後 — **暗影裂隙與梅林流落** — 巫師戰爭驚醒暗影惡魔，梅林被吸入暗影界 → [詳見](events.md#暗影裂隙與梅林流落the-shadow-rift)
- SM 背景 — **佛比安帝國崛起** — 佛比烏斯以反魔法之名建帝國 → [詳見](events.md#佛比安帝國崛起rise-of-the-phobian-empire)
- QJR 382 — ★ **暗影惡魔之戰／擊敗萬噬者** — 梅林率六方擊潰萬噬者，億萬靈魂獲釋 → [詳見](events.md#暗影惡魔之戰擊敗萬噬者defeat-of-the-all-devourer)

## 第三紀：聯邦時代（AoW3 前傳背景）

> SM 之後茱莉亞重塑精靈、創立法治聯邦；本作三部曲不演此段，僅作後續指引。

- QJR 150 — **彌合儀式** — 茱莉亞與薩里達斯重造高等精靈 → [詳見](events.md#彌合儀式the-mending)
- QJR 384–394 — **暗影飛遁、和平十年、聯邦籌建** — 賽隆與執政官追剿暗影惡魔，米安多與茱莉亞共治
- YC 1（QJR 395） — ★ **聯邦成立** — 諸王議會定《治理法》，第三紀開始 → [詳見](events.md#聯邦成立founding-of-the-commonwealth)
- YC 131–136 — **米安多歸來與大浩劫** — 米安多自暗影門出兵、梅林摧毀最後暗影門，奪去諸巫王之力
- YC 225 — **AoW3 戰役開始** — 聯邦與精靈宮廷最後談判破裂（超出本三部曲範圍）
