# 《斯露德》編年表

## 第一紀：碧空時代（前異生體時期）

> 人類統治天空與大地的繁榮時期；因異生體的突然入侵而終結。

- [起點] — **人類文明巔峰** — 科學與符能高度發展，天空通行無阻。
- [紀末] — **異生體入侵** — 神祕敵對生物突然出現在天空，人類失去統治權 → [詳見](events.md#異生體入侵)

## 第二紀：遮蔽時代（末世避難時期）

> 人類退縮至「帆」等避難所，依靠「遮蔽層」苟延殘喘。

- [入侵初期] — **遮蔽層建立** — 為了保護最後的人類居住區，神祕屏障建立 → [詳見](events.md#遮蔽層建立)
- [中期] — **斯露德技術誕生** — 人類發現能與源能產生共鳴的少女，開啟反擊希望。
- [當前] — ★ **故事主線開始** — 導師加入瓦哈納艦隊，展開奪回天空的航程。
- [時期] — **溫德蘭德防線** — 導師帶領斯露德抵擋異生體入侵 → [詳見](events.md#邁向毀滅的起點)
- [時期] — **波伽蘭德碎石事件** — 瓦希緹上將回歸，守護城市平民 → [詳見](events.md#天將傾覆人歸何處)
- [時期] — **拉普蘭極地調查** — 揭示斯露德與異生體能量同源的殘酷真相 → [詳見](events.md#滯鎖極境)
- [近期] — **龍城事件** — 抵達東方龍城，揭發人類高層的長期隱瞞 → [詳見](events.md#龍城篇)

## 第三紀：重返藍天（未來展望）

> 人類揭開世界真相，斯露德們為了自由與未來而戰。

- [展望] — **破除遮蔽層** — 揭露真相後的終極目標。
- [展望] — **重建世界秩序** — 在導師與斯露德的共同努力下，人類重返天空。
