# 《瑪娜希斯回響》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物
- [主角 / 助手](characters.md#主角--助手-protagonist) — 玩家扮演的箱庭島分院學院長助手，故事的見證者與參與者。
- [塞納菲娜·阿什克羅夫特](characters.md#塞納菲娜阿什克羅夫特-serafina-ashcroft) — 來自東方精靈村落的法師少女，箱庭島分院的核心學生之一。
- [米爾德里德·彌爾蒂](characters.md#米爾德里德彌爾蒂-mildred-mirti) — 南方劍士家族彌爾蒂的繼承人，性格嚴謹的戰士少女。
- [吉爾伯塔·梅普爾](characters.md#吉爾伯塔梅普爾-gilberta-maple) — 瑪娜希斯教會撫養長大的牧師少女，性格冷靜沉穩。
- [法露露·卡普安納](characters.md#法露露卡普安納-faruru-capuana) — 森之精靈族弓箭手，性格極度自由且嗜睡。
- [伊麗莎·弗洛迪斯](characters.md#伊麗莎弗洛迪斯-elisa-flodis) — 北海武裝商船團團長的千金，帶有大姐姐氣質的學生。
- [米埃爾·哈奈特](characters.md#米埃爾哈奈特-miel-harnett) — 西部沙漠地下都市出身的龍人族少女。
- [薩拉·阿貝爾](characters.md#薩拉阿貝爾-sara-abel) — 追尋「槍之極道」的武術少女。
- [格麗澤爾達·麥克威廉姆斯](characters.md#格麗澤爾達麥克威廉姆斯-grizelda-mcwilliams) — 西奧德里克學院箱庭島分院的學院長。
- [蓋茨·西格特](characters.md#蓋茨西格特-gates-sigert) — 學院武技課教師，熱血豪爽的戰士。
- [魯納莉卡·馮頓海姆](characters.md#魯納莉卡馮頓海姆-lunalica-von-fonheim) — 學院採集課教師，負責野外教學。
- [艾瑞莎·德拉伍德伯](characters.md#艾瑞莎德拉伍德伯-arisa-delawoodber) — 學院奧秘課教師，精靈煉金術專家。
- [喵姆·芙洛喵](characters.md#喵姆芙洛喵-nyumu-flonia) — 學院社會課教師，貓型獸人。
- [西貝爾·豪爾贊](characters.md#西貝爾豪爾贊-siebel-hawzen) — 學院冒險課教師，男裝麗人。
- [雅爾托妮婭](characters.md#雅爾托妮婭-yartonia) — 冒險者公會派遣駐校的公務員。
- [史蒂希婭](characters.md#史蒂希婭-sticia) — 金匠學院學生，阿爾扎德家族成員。
- [格雷布](characters.md#格雷布-gleb) — 與史蒂希婭相關的具名男性角色。
- [西奧德里克](characters.md#西奧德里克-theodoric) — 傳說中的勇者，弗多尼亞大陸的守護者。

## 種族 / 陣營
- [西奧德里克學院](factions.md#西奧德里克學院-theodoric-academy) — 由勇者西奧德里克創立，旨在培育冒險者與監視魔王力量的綜合教育機構。
- [箱庭島分院](factions.md#箱庭島分院-hakoniwa-branch) — 位於世外桃源「箱庭島」上的學院分部，是本作故事的主要發生地。
- [瑪娜希斯教會](factions.md#瑪娜希斯教會-manasis-church) — 崇尚瑪娜希斯（精靈）力量並在世界各地提供救濟的宗教組織。
- [冒險者公會](factions.md#冒險者公會-adventurers-guild) — 管理弗多尼亞大陸上所有冒險者委託與等級評定的專業組織。
- [金匠學院](factions.md#金匠學院-goldsmith-academy) — 位於弗多尼亞北方的一所知名冒險者學院。
- [阿爾扎德家族](factions.md#阿爾扎德家族-alzad-family) — 弗多尼亞南方的名門望族，擁有深厚的財力與政治影響力。
- [北海武裝商船團](factions.md#北海武裝商船團-north-sea-armed-merchant-fleet) — 活躍於北海海域，兼具貿易與武裝防衛能力的船團。
- [龍人族](factions.md#龍人族-dragonewts) — 擁有龍族血統，具備探查世界能量流動能力的少數種族。

## 事件
- [精靈戰爭](events.md#精靈戰爭-manasis-war) — 遠古時代發生在精靈（瑪娜希斯）之間的混戰。
- [魔王戰爭](events.md#魔王戰爭-demon-king-war) — 勇者西奧德里克帶領夥伴對抗魔王的決定性戰爭。
- [封印魔王](events.md#封印魔王) — 結束魔王戰爭的關鍵行動，亦是傳說的終點。
- [第七章：嚴寒地對抗](events.md#第七章嚴寒地對抗-chapter-7) — 遊戲停運前推進到的最後一個主線大事件。

## 地點
- [弗多尼亞](places.md#弗多尼亞-ferdonia) — 故事發生的核心主大陸。
- [箱庭島](places.md#箱庭島-hakoniwa-island) — 學院所在地，玩家活動的核心區域。
- [東方精靈村落](places.md#東方精靈村落) — 塞納菲娜的故鄉。
- [里卡薩](places.md#里卡薩-ricasa) — 南方的港灣都市，劍士家族彌爾蒂的根據地。
- [西部沙漠地下都市](places.md#西部沙漠地下都市) — 龍人族生活的沙漠區域。
- [北海](places.md#北海-north-sea) — 伊麗莎故鄉所在的海域。
- [嚴寒之地](places.md#嚴寒之地-northern-frozen-lands) — 大陸北方的極寒區域，主線第七章的主要發生地。
- [阿蘇勒遊樂園](places.md#阿蘇勒遊樂園-asule-amusement-park) — 遊戲後期（官方維護更新）新增的純休閒區域。
- [精靈樹](places.md#精靈樹-manasis-tree) — 學院內蘊含極高濃度瑪娜的巨大古樹。

## 概念 / 設定
- [瑪娜希斯](concepts.md#瑪娜希斯-manasis--精靈) — 世界力量的源頭，亦是遠古戰爭的引發者。
- [魔王](concepts.md#魔王-demon-king) — 誕生自精靈戰爭的惡意聚合體。
- [瑪娜](concepts.md#瑪娜-mana) — 構建世界、驅動術法與生命運作的基本能量。
- [龍脈](concepts.md#龍脈-dragon-veins) — 地底流動的世界能量大動脈。
- [海魔之眼](concepts.md#海魔之眼) — 弗洛迪斯家族傳承的血脈詛咒。
- [元素系統](concepts.md#元素系統-elemental-system) — 世界運作的六大屬性法則。
