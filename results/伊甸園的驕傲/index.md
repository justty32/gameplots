# 《伊甸園的驕傲》索引

## 人物
- [團長 (Leader)](characters.md#團長-leader--protagonist) — 從地球轉生至伊甸大陸的人類，Pride 冒險團指揮官。
- [女神 伊萬斯 (Evans)](characters.md#女神-伊萬斯-evans--evante) — 引導團長轉生的懶散女神，看板娘之一。
- [阿普里爾 (April)](characters.md#阿普里爾-april) — Pride 冒險團初期夥伴，可靠的先鋒。
- [奧利維亞 (Olivia)](characters.md#奧利維亞-olivia) — 核心戰力，連結主角與大陸歷史的橋樑。
- [諾埃爾 (Noel)](characters.md#諾埃爾-noel) — 海兔原型 RED，外表溫婉戰場破壞力驚人。
- [拉比 (Rabi)](characters.md#拉比-rabi) — 雪兔原型 RED，高機動爆發型輸出。
- [莎拉 (Sarah)](characters.md#莎拉-sarah) — 黑豹原型 RED，精準反擊型戰士。
- [奎恩 (Quinn)](characters.md#其他重要成員) — 孔雀原型，高傲華麗。
- [莫妮卡 (Monica)](characters.md#其他重要成員) — 野豬原型，衝勁十足。
- [尤娜 (Yuna)](characters.md#其他重要成員) — 獨角鯨原型，神祕輔助。
- [卡農 (Canon)](characters.md#其他重要成員) — 烏鴉原型，偵查支援。

## 種族 / 陣營
- [RED](factions.md#red-refined-exceptional-daughters) — 覺醒動物特徵與異能的少女群體，社會邊緣的核心戰力。
- [Pride](factions.md#pride-驕傲冒險團) — 由 RED 組成、人類擔任團長的合法冒險組織。
- [艾爾雅拉特聖王國](factions.md#艾爾雅拉特聖王國-el-arat-holy-kingdom) — 大陸最強軍事強權。
- [羅賽莉亞教皇國](factions.md#羅賽莉亞教皇國-roselia-holy-empire) — 以宗教守護聖遺物的強權。
- [艾爾雅拉特聖騎士團](factions.md#艾爾雅拉特聖騎士團) — 聖王國精銳軍事力量。
- [伊甸騎士](factions.md#伊甸騎士-eden-knights) — 與聖遺物訂立契約的特殊戰士。

## 事件
- [那場戰爭](events.md#那場戰爭-the-war) — 導致 RED 地位一落千丈的歷史關鍵衝突。
- [主角轉生](events.md#主角轉生) — 故事的起源，跨越世界線的轉折。
- [尋找那隻貓](events.md#尋找那隻貓) — 故事的核心驅動力，返鄉之路的鑰匙。

## 地點
- [伊甸大陸](places.md#伊甸大陸-eden-continent) — 人類與 RED 並存的異世界舞台。
- [艾爾雅拉特聖王國領域](places.md#艾爾雅拉特聖王國領域) — 大陸最強人類國家的核心領土。
- [羅賽莉亞教皇國領域](places.md#羅賽莉亞教皇國領域) — 宗教強權與聖遺物的守護地。
- [雪原地帶](places.md#雪原地帶) — 拉比的故鄉，極端環境的 RED 生存地。
