# 《幻象回憶》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物 (Characters)
- [御主（Master）](characters.md#御主master) — 玩家扮演的主角，被稱為「救世主」，負責召喚並指揮幻象從者。
- [加百列（Gabriel）](characters.md#加百列gabriel) — 四大天使長之首，對抗路西法的核心領袖，時空之門的守護者。
- [路西法（Lucifer）](characters.md#路西法lucifer) — 墮天使，本作主要反派，撕裂現實的叛逆者。
- [哈迪斯（Hades）](characters.md#哈迪斯hades) — 冥界之主，具有極強的女王氣場，掌管生死與地底資源。
- [漢摩拉比（Hammurabi）](characters.md#漢摩拉比hammurabi) — 古巴比倫之王，代表法律與正義的執行者，具備霸氣大姊姊性格。
- [雅典娜（Athena）](characters.md#雅典娜athena) — 戰爭與智慧女神，充滿行動力與正義感的少女。
- [撒拉丁（Saladin）](characters.md#撒拉丁saladin) — 騎士精神的化身，手持法杖守護和平的光輝之神。
- [賽特（Seth）](characters.md#賽特sety) — 風暴之神，擁有絕對破壞力量的強力戰士。
- [女媧（Nuwa）](characters.md#女媧nuwa) — 東方神話的大地女神，象徵創造與慈愛。

## 種族 / 陣營 (Factions)
- [幻象從者（Mirage）](factions.md#幻象從者mirage) — 由歷史與神話英雄女性化後的從者群體，本作的核心戰鬥力量。
- [拉米拉皮斯勢力（Ramirapis Forces）](factions.md#拉米拉皮斯勢力ramirapis-forces) — 遺忘之境的守護與統治勢力，代表原本的神之秩序。
- [墮落勢力（Fallen Forces）](factions.md#墮落勢力fallen-forces) — 由路西法領導的反叛組織，企圖推翻神之統治。

## 事件 (Events)
- [人類滅亡](events.md#人類滅亡) — 發生於故事主線遙遠過去的背景事件。
- [路西法之亂與現實撕裂](events.md#路西法之亂與現實撕裂) — 路西法對神之統治發起的武裝叛變，導致世界破碎的關鍵事件。
- [御主降臨與記憶尋回](events.md#御主降臨與記憶尋回) — 玩家覺醒於遺忘之境的轉折點，也是主線的起點。

## 地點 (Places)
- [遺忘之境（Lethe）](places.md#遺忘之境lethe) — 人類滅亡後的異世界舞台。
- [後宅（Backyard）](places.md#後宅backyard) — 供御主與幻象從者生活與交流的私密空間。

## 概念 / 設定 (Concepts)
- [幻象（Mirage）](concepts.md#幻象mirage) — 歷史英靈在遺忘之境重現的生命形態，實為「DNA 記憶」的實體化。
- [DNA 記憶（DNA Memory）](concepts.md#dna-記憶-dna-memory) — 連結古代神話與未來末世的核心科幻設定。
- [時空之門（Time Gate）](concepts.md#時空之門-time-gate) — 遺忘之境中連結不同維度的關鍵樞紐。
- [遺忘之境（Lethe）](concepts.md#遺忘之境lethe) — 魔法與現實交織的異世界，建立在人類文明殘骸之上。
- [50 vs 50 世界戰](concepts.md#50-vs-50-世界戰) — 幻象從者間的大規模軍團對戰系統。
