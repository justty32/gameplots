# 《東京戰記 (東京喰種)》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物
- [金木研 / 佐佐木琲世](characters/Kaneki_Ken.md) — 本作主角，從人類轉變為半喰種，後成為「獨眼之王」與「龍」。
- [霧嶋董香](characters/Kirishima_Touka.md) — 女主角，安定區咖啡廳服務生，代號「兔子」，金木之妻。
- [芳村功善](characters/Yoshimura.md) — 安定區店長，SSS級喰種「不殺之梟」。
- [四方蓮示](characters.md#四方蓮示-yomo-renji) — 店長助手，代號「烏鴉」，實力強大且沉默寡言。
- [笛口雛實](characters.md#笛口雛實-fueguchi-hinami) — 父母雙亡的少女喰種，視金木與董香為親人。
- [西尾錦](characters.md#西尾錦-nishio-nishiki) — 藥學系大學生，後成為代號「大蛇」的強大喰種。
- [入見佳耶](characters.md#入見佳耶-irimi-kaya) — 咖啡廳店員，原「黑色多伯曼」首領。
- [古間圓兒](characters.md#古間圓兒-enji-koma) — 咖啡廳店員，原「魔猿」首領。
- [有馬貴將](characters/Arima_Kisho.md) — CCG特等搜查官，「白色死神」，前任獨眼之王。
- [亞門鋼太朗](characters/Amon_Kotaro.md) — 正義感極強的搜查官，後被改造為半喰種「軟盤」。
- [鈴屋什造](characters/Suzuya_Juuzou.md) — 戰鬥天賦異稟的特等搜查官，性格怪異但重情義。
- [真戶曉](characters.md#真戶曉-mado-akira) — 真戶吳緒之女，冷靜睿智，曾任佐佐木的上司。
- [真戶吳緒](characters.md#真戶吳緒-mado-kureo) — 亞門的搭檔，對庫因克極度瘋狂，後被董香殺害。
- [瀧澤政道](characters.md#瀧澤政道-seidou-takizawa) — 搜查官，後被改造為瘋狂的半喰種「貓頭鷹」。
- [艾特 / 高槻泉](characters/Eto.md) — 青銅樹首領，「獨眼之梟」，知名的恐怖小說家。
- [多多良](characters.md#多多良-tatara) — 青銅樹幹部，冷酷的領導者，來自中國。
- [野呂](characters.md#野呂-noro) — 具備超常再生能力的幹部。
- [霧嶋絢都](characters.md#霧嶋絢都-kirishima-ayato) — 董香的弟弟，代號「黑兔」，性格好戰。
- [壁虎 / 大守八雲](characters.md#壁虎--大守八雲-yamori) — 13區殘暴喰種，啟發金木覺醒的虐待者。
- [舊多二福](characters/Furuta_Nimura.md) — 故事後期的最大幕後黑手，野心家與破壞者。
- [唄](characters.md#唄-uta) — 面具師，代號「無臉」，小丑成員。
- [神代利世](characters.md#神代利世-kamishiro-rize) — 故事起點，其器官移植給了金木，被稱為「大食客」。
- [月山習](characters.md#月山習-tsukiyama-shuu) — 喰種美食家，對金木的肉質有著病態的執著。
- [永近英良](characters.md#永近英良-hide) — 金木唯一的親友，為了守護金木不惜一切。
- [瓜江久生](characters.md#瓜江久生-kuki-urie) — Qs班首任班長，性格功利但在戰鬥中不斷成長。
- [不知吟士](characters.md#不知吟士-ginshi-shirazu) — Qs班次任班長，為了守護同伴而戰死的英雄。
- [米林才子](characters.md#米林才子-saiko-yonebayashi) — 宅女屬性的天才喰種，Qs班的溫暖核心。
- [六月透](characters.md#六月透-tooru-mutsuki) — 內向但內心扭曲的成員，對金木有病態的愛。

## 種族 / 陣營
- [喰種](factions.md#喰種-ghoul) — 以人類為食的亞人種。
- [半喰種](factions.md#半喰種-half-ghoul) — 人類與喰種的混血或通過人工器官移植產生的存在。
- [CCG](factions.md#ccg-喰種對策局--commission-of-counter-ghoul) — 專門驅逐、研究喰種的國家機關。
- [安定區](factions.md#安定區-antei-ku--20區咖啡廳) — 20區的和平派喰種組織。
- [青銅樹](factions.md#青銅樹-aogiri-tree) — 由獨眼之梟創立的激進喰種武裝組織。
- [小丑](factions.md#小丑-clowns) — 追求混亂與享樂的神秘組織。
- [V](factions.md#v) — 隱藏在幕後操縱世界平衡的組織。
