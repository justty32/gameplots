# 赫子 (Kagune) 與 Rc 細胞物理學

> 喰種的戰鬥與捕食器官，存在一套嚴謹的生物物理學與相剋鏈系統。

- **本質**：由體內高濃度的 **Rc 細胞** 突破皮膚後凝固而成的捕食器官。其形態與性質完全取決於 Rc 細胞的**結合方式**與**釋放速率**。
- **故事中的功能**：不僅是喰種捕食與自保的手段，也是 CCG 製作武器「庫因克」的原料。赫子的強弱與形態象徵了角色的個性與生存哲學。

### 核心分類與生物特性
1. **羽赫 (Ukaku)**：集中於肩胛骨附近。高頻率釋放，細胞結合較鬆散。追求絕對的速度與遠程爆發，但**持久力極差**。
2. **甲赫 (Koukaku)**：集中於肩胛骨下方。高密度結合，細胞排列極其緊密。追求極致的硬度與防禦，但**重量大、動作遲緩**。
3. **鱗赫 (Rinkaku)**：集中於腰部附近。細胞結合呈鱗片狀。追求強大的打擊力與**卓越的再生能力**，但結構較為**脆弱**。
4. **尾赫 (Bikaku)**：集中於尾椎附近。細胞結合最為均衡。追求穩定性，攻防一體，**沒有明顯短板，但也缺乏極端優勢**。

### 物理與戰術的博弈（相剋原理）
石田翠設定的相剋循環（羽 → 尾 → 鱗 → 甲 → 羽）並非玄學，而是基於能量消耗、硬度與機動力的物理平衡：
- **羽赫 剋 尾赫 (速度壓制平衡)**：羽赫的遠程地毯式轟炸能輕易消耗掉試圖接近的尾赫。
- **尾赫 剋 鱗赫 (穩定擊碎脆弱)**：尾赫紮實的物理強度能精準切斷鱗赫脆弱的根部，讓其再生速度趕不上破壞速度。
- **鱗赫 剋 甲赫 (穿透瓦解剛硬)**：鱗赫強大的瞬間爆發力如鑿子般反覆擊打同一點，最終導致高密度的甲赫產生裂紋崩碎。
- **甲赫 剋 羽赫 (耐久耗死爆發)**：甲赫只需縮小防禦範圍擋住結晶彈，一旦羽赫進入疲勞期，沉重的甲赫就能輕易收割。

### 哲學象徵：捕食者的生態圈
赫子的相剋原理象徵著世界觀中**「沒有絕對的強者」**的主題：
羽赫是「剎那的輝煌」；甲赫是「孤獨的堡壘」；鱗赫是「瘋狂的再生」；尾赫是「中庸的智慧」。這種沒有頂端的相剋鏈，完美註解了《東京喰種》中「每個人都是捕食者，也都是被捕食者」的悲劇性生存競爭。
