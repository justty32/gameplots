# 《幻想神域》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物
- [啟源使者](characters.md#啟源使者-envoy-of-gaia) — 玩家扮演的主角，被「啟源方塊」選中的救世者。
- [萊因哈特 / 黑騎士](characters.md#萊因哈特-reinhart--黑騎士-black-knight) — 前聖堂騎士團長，現為惡魔軍團首領，故事的核心反派。
- [蘇娜](characters.md#蘇娜-suna) — 萊因哈特之女，現為優秀的聖堂騎士，玩家的重要夥伴。
- [亞伯拉罕](characters.md#亞伯拉罕-abraham) — 前任教宗。
- [艾瑪](characters.md#艾瑪-emma) — 現任教宗。
- [露西亞](characters.md#露西亞-lucia) — 萊因哈特之妻，蘇娜之母。
- [亞當](characters.md#亞當-serif) — 代表「力量」的初始源神。
- [夏娃](characters.md#夏娃-merrilee) — 代表「生命」的初始源神。
- [莉莉絲](characters.md#莉莉絲-lilith) — 代表「黑暗與誘惑」的強大源神。
- [凱薩](characters.md#凱薩-kaiser) — 代表「勇氣」的鐵血戰士型源神。

## 種族 / 陣營
- [聖堂](factions.md#聖堂-the-holy-hall) — 維護人界秩序的正義與信仰中心。
- [惡魔軍團](factions.md#惡魔軍團-the-demons) — 來自地獄的侵略勢力。
- [獅族](factions.md#獅族-lionfolk) — 居住在野外、體格強健的亞人種族。
- [蛙人](factions.md#蛙人-frogmen) — 生活在水邊或濕地的奇特種族。
- [矮人](factions.md#矮人-dwarves) — 擅長工藝與建築的種族。
