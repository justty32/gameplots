# 阿卡迪亞 - 物品詞條 (Items)

---

## 神力遺物

### 聖印 (Holy Seals)

> 倉頡知道自己無法永遠守護——所以他把力量留了下來，等待繼任者。

- **描述**：由創世帝王倉頡在達沃爾恩帝國鼎盛時期親手設計的神性結晶系統。倉頡以自身天生覺醒的完整神格，提取各類神格的精華，將其封存為可傳承的物質形態。
- **分布**：聖印節點散布於鏡之大陸各地，部分在已知的遺跡中，部分被各大勢力掌握，部分深埋在須佐家族守護的封印森林深處。
- **功能**：
  - 強化神之子的神格能力，解鎖超越一般覺醒者能力的特殊技能
  - 作為大型封印陣列的「錨點」，維持對虛無（外神威脅）的封印結構
  - 傳說集齊所有聖印的人將能重現倉頡的完整神格
- **各國態度**：三大勢力（刀鋒帝國、費羅斯、達沃爾恩）都在積極蒐集聖印，但目的各異——刀鋒帝國要用它強化戰力，費羅斯要研究其機制，達沃爾恩要以倉頡的名義主張正統繼承。
- **最終用途（劇透）**：終盤揭露，集齊所有聖印並重建封印陣列，是阻止虛無侵蝕完全突破的唯一方法。這也意味著聖印不是「賞給英雄的寶物」，而是「倉頡預留的守護機制」。

### 神格之源 (Godhood Core / 神格核心)

> 不是所有力量都在等待被找到——有些力量在等待足夠強大的人來承受它。

- **描述**：隱藏在鏡之大陸各處、尚未被認領的原生神格碎片。與聖印不同，神格之源是「神明意志的殘留」，而非倉頡人工封印的設計品。
- **性質**：每個神格之源對應一個特定的古代神明（如波塞冬、阿波羅、阿瑞斯等），且只有體質匹配的個體才能成功接觸而不被反噬。
- **取得風險**：強行接觸不匹配的神格之源，可能引發「魔物化」——無法負荷的神性從內部瓦解宿主的靈魂。
- **政治意義**：各勢力爭奪神格之源，是引發局部戰爭的直接導火線。費羅斯對神格之源的「適配研究」是其最敏感的科研方向。

## 戰術工具

### 指揮官戰術矩陣 (Commander's Tactical Matrix / 3x3 戰術盤)

- **描述**：指揮官所使用的特製戰場佈陣系統，以九宮格形態讓神之子在戰場上形成特定的站位陣列。
- **設計來源**：其基礎原理源自倉頡時代達沃爾恩帝國的「神能共鳴陣」研究，由費羅斯聯盟的學者後來加以改良並提供給指揮官使用。
- **敘事意義**：在劇情中，戰術矩陣不只是戰鬥系統，而是「指揮官與神之子的神能共振場」——正因指揮官自身不具備神格，他/她需要這個工具來與神之子的力量產生協同效應。
- **象徵**：九宮格的中心位置是空的——指揮官站在中心，用非神格的人性意志串聯起所有神格力量。這個設計本身是倉頡留給未來指揮者的啟示：**神格的力量需要人性的智慧來引導，而不是相反。**

## 達沃爾恩遺物

### 倉頡的語言記錄 (Cang Jie's Writing Records / 倉頡的文字)

> 不是普通的文字——每一個符文都是神格的容器。

- **描述**：倉頡作為以「文字與知識」為工具建立帝國的神之子，留下了大量以特殊符文書寫的記錄。這些文字不只是普通的語言，而是能在特定條件下激活神格共鳴的「活性符文」。
- **分布**：達沃爾恩古代聖印庫（Ancient Seal Archive）保存了最完整的版本，但在帝國崩潰時部分資料流失，副本散布在費羅斯和刀鋒帝國手中。
- **費羅斯的解讀研究**：費羅斯聯盟對倉頡文字的研究是其最重要的學術資產，也是他們理解神格機制最主要的途徑。

### 封印陣列節點 (Sealing Array Nodes)

- **描述**：倉頡在鏡之大陸各地設置的封印結構組成元素，每個節點以特定的聖印為核心，彼此之間形成覆蓋整個大陸的封印網絡。
- **現狀**：由於倉頡的失蹤，封印陣列一直以最低限度的自動模式運行，而非全功率維持。部分節點因無人維護而開始衰退，導致「虛無」的滲透速度逐年加快。
- **須佐家族的職責**：守護封印森林深處的多個重要節點，是他們千年傳承最核心的使命。
- **最終任務**：主線終盤，重建完整的封印陣列是阻止虛無入侵的唯一方法，而這需要所有神之子共同參與——以及指揮官找到每一個節點的精確位置。
