# 阿卡迪亞 - 事件詞條 (Events)

### 倉頡失蹤
- **描述**：古帝國鼎盛時期，第一位神之子倉頡突然在宮廷中消失，未留下繼承人。
- **影響**：這是鏡之大陸墮入亂世的轉折點。由於失去了最強大的神力威壓，各方軍閥與野心家開始分裂帝國，導致長達數百年的混亂。

### 神之子覺醒潮
- **描述**：在連年戰亂中，大陸各地開始出現大量的「神之子」覺醒現象。
- **背景**：普通人在瀕死或絕望之際與古代神格產生共鳴，獲得異能。這改變了戰爭的型態，使得個體英雄的力量足以左右戰場勝負。

### 達沃爾恩大崩潰
- **描述**：伴隨倉頡失蹤而來的帝國行政與防禦體系崩壞。
- **結果**：大量古代科技與聖印資料流失，部分落入刀鋒帝國與費羅斯之手，形成今日鼎足而立的局勢。

### 聖印爭奪戰
- **描述**：圍繞著倉頡留下的強化遺產「聖印」所展開的各類規模衝突。
- **敘事意義**：遊戲主線的核心推動力，玩家透過這些戰役逐步收集碎片，重建倉頡的力量。
