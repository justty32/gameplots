# 《黑色沙漠 (Black Desert)》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物
- [冒險家 (Adventurer)](characters.md#冒險家-adventurer) — 玩家扮演的角色，失去記憶的古代文明後裔，天生的英雄。
- [黑精靈 (Black Spirit)](characters.md#黑精靈-black-spirit) — 伴隨玩家的神秘寄生體，引導冒險家尋找力量的「影子」。
- [哈杜姆 (Hadum)](characters.md#哈杜姆-hadum) — 絕望與黑暗之神，異界「哈杜姆領域」的統治者。
- [謝勒汗 (Sherekan)](characters.md#謝勒汗-sherekan) — 德利勘傳說中的英雄戰士，曾帶領族人對抗巨龍。
- [希爾比亞 (Sylvia)](characters.md#希爾比亞-sylvia) — 精靈的創造女神，卡瑪希爾夫聖樹的賦予者。
- [阿希伏 (Ahib)](characters.md#阿希伏-ahib) — 指代被驅逐的影子精靈勢力，現為奧迪爾利塔的統治階級。

## 種族 / 陣營
- [卡爾佩恩共和國 (Calpheon)](factions.md#卡爾佩恩共和國-calpheon) — 西方最強大的商業強權，採取共和體制的霸權國家。
- [瓦倫西亞王國 (Valencia)](factions.md#瓦倫西亞王國-valencia) — 佔據東方廣大沙漠區域的古老文明國家，守護著沙漠深處的遺址。
- [賽林迪亞 (Serendia)](factions.md#賽林迪亞-serendia) — 位於地圖中心，資源豐富但飽受戰爭蹂躪的領地。
- [巴雷諾斯 (Balenos)](factions.md#巴雷諾斯-balenos) — 和平的沿海地區，冒險家故事的起點。
- [梅迪亞 (Mediah)](factions.md#梅迪亞-mediah) — 位於卡爾佩恩與瓦倫西亞交界，掌握黑石加工技術的新興國家。
- [卡瑪希爾比亞 (Kamasylvia)](factions.md#卡瑪希爾比亞-kamasylvia) — 精靈族居住的森林國度，擁有與自然共生的古老文化。
- [德利勘 (Drieghan)](factions.md#德利勘-drieghan) — 險峻山岳區域，以狩龍文化與堅毅戰士聞名的領地。
- [奧迪爾利塔 (O'dyllita)](factions.md#奧迪爾利塔-odyllita) — 由被驅逐的阿希伏精靈所建立的國度，隱藏於黑暗與絕望中。
- [晨曦之國](factions.md#晨曦之國) — 位於極東方的古老國度，充滿東方神怪色彩。

## 事件
- [黑死病爆發](events.md#黑死病爆發) — 由瓦倫西亞商人意外傳入西方的災難性瘟疫。
- [十字軍遠征 (黑石戰爭)](events.md#十字軍遠征-黑石戰爭) — 卡爾佩恩假借宗教之名，向瓦倫西亞發動的資源爭奪戰。
- [古代文明滅亡](events.md#古代文明滅亡) — 曾擁有高度黑石提煉技術的文明一夕之間消失的事件。

## 地點
- [黑色沙漠 (Black Desert) / 紅色沙漠 (Red Desert)](places.md#黑色沙漠-black-desert--紅色沙漠-red-desert) — 同一片沙漠的兩個名稱，象徵了不同國家的史觀與立場。
- [海地爾 (Heidel)](places.md#海地爾-heidel) — 賽林迪亞的首府，充滿歐式風情的河畔商業都市。
- [卡瑪希爾夫 (Kamasylve)](places.md#卡瑪希爾夫-kamasylve) — 卡瑪希爾比亞的神聖聖樹，精靈族的生命起源。
- [永冬之山](places.md#永冬之山) — 終年積雪的高聳山脈，傳說中守護著古代聖火。

## 物品 / 技術
- [黑石 (Black Stone)](items.md#黑石-black-stone) — 古代文明的動力核心，呈現為黑色結晶狀的高能物質。
- [聖火](items.md#聖火) — 傳說中能驅散哈杜姆黑暗、燃燒一切邪惡的神聖火焰。

## 概念 / 設定
- [艾爾利恩教](concepts.md#艾爾利恩教) — 西方諸國（卡爾佩恩、賽林迪亞）的主流宗教，信仰光之神艾爾利恩。
- [阿爾信仰](concepts.md#阿爾信仰) — 瓦倫西亞的唯一神信仰，崇拜神之使者阿爾。
- [知識系統 (Knowledge System)](concepts.md#知識系統-knowledge-system) — 遊戲的核心機制，代表玩家對世界的理解程度。
- [哈杜姆的侵蝕](concepts.md#哈杜姆的侵蝕) — 黑暗之神試圖從異界滲透並吞噬現世的現象。
