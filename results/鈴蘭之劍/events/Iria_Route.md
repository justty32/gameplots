# 伊利亞王國線 (Iria Route)

> 事件類型：多分支主線劇情 / 政治權謀與亡國史詩

- **時間**：命運螺旋第 19 週之後
- **參與者**：主角（傭兵團長）、[丹塔里恩](../characters/Dantalion.md)、[伊南娜](../characters/Inanna.md)、[索菲亞](../characters.md#索菲亞-sofia)
- **進入條件**：初期面對威脅時，選擇向「伊利亞王國軍」求助，並護送伊南娜回到晨曦堡。

### 核心衝突：鐵腕救國的現實主義與純潔理想的碰撞

伊利亞線是一部關於「犧牲」與「傳承」的悲劇史詩。核心衝突在於「大王子迪塔利奧的現實主義」與「三王女伊南娜的理想主義」的碰撞。
迪塔利奧背負了國家的黑暗，他深愛伊利亞，但認為在列強環伺下，只有通過冷酷的手段、甚至犧牲部分人民才能換取國家的獨立。伊南娜則象徵著伊利亞的良知，試圖用溫柔感化世界。玩家夾在兩者之間，親歷一個國家在絕境中的掙扎。

### 關鍵結局分支

#### 1. 結局一：【和平的曙光】（真結局 / True End）
- **達成條件**：需完成多項前置民意任務與特定戰鬥（如礦山城不撤退、炸彈設置等），最關鍵的是**伊南娜演說成功**。
- **經過與結果**：伊南娜通過真誠的演說喚醒了民眾的鬥志與團結。迪塔利奧意識到妹妹已經成長為合格的領袖，選擇隱退或自我犧牲。伊利亞在他的佈局與伊南娜的感召下，成功在列強的夾縫中生存下來，實現了真正的獨立。
- **敘事意義**：如果只有黑暗（迪塔利奧），國家會崩潰；如果只有理想（伊南娜），國家會被吞併。只有當主角介入，讓理想擁有了力量，迪塔利奧的犧牲才有了真正的價值。

#### 2. 結局二：【無冕的守護者】（普通結局 / Normal End）
- **達成條件**：伊南娜演說失敗，但在最後關頭選擇「與伊南娜一同留下」。
- **經過與結果**：局勢崩潰，迪塔利奧為了大局犧牲自己。主角與伊南娜在廢墟上艱難重建。主角成為了沒有王冠的守護者。
- **敘事意義**：保住了國家，但代價極其慘重，充滿了戰爭過後的荒涼感。

#### 3. 結局三：【毀滅與逃亡】（壞結局 / Bad End）
- **達成條件**：伊南娜演說失敗，且在最後關頭選擇「帶著伊南娜逃走」。
- **經過與結果**：晨曦堡陷落，迪塔利奧戰死。主角帶著伊南娜流亡海外。伊利亞王國徹底滅亡，淪為列強的殖民地。
- **敘事意義**：充滿遺憾的結局，國家失去了主權，希望之火熄滅，主角只能作為歷史的旁觀者逃離。

### 哲學探討
伊利亞線深入描繪了被「資源詛咒」的弱國悲歌。迪塔利奧是一個極具悲劇色彩的梟雄，他不怕弄髒自己的雙手，卻始終得不到理解。這條線質問玩家：為了一個國家的存亡，道德的底線究竟可以退讓到什麼地步？
