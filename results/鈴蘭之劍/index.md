# 《鈴蘭之劍：為這和平的世界》索引

> 本作小百科的入口。要 5 分鐘讀懂全貌請看 [synopsis.md](synopsis.md)；要按時間順序看事件請看 [timeline.md](timeline.md)。

## 人物
- [拉維耶 (Rawiyah)](characters.md#拉維耶-rawiyah) — 鈴蘭之劍傭兵團創始人，性格豪爽、武藝超群的引路人。
- [麥莎 (Maitha)](characters.md#麥莎-maitha) — 傭兵團護盾手，開朗樂觀的初始夥伴。
- [法卡爾 (Faycal)](characters.md#法卡爾-faycal) — 傭兵團弓箭手，性格冷靜。
- [丹塔里恩 (Dantalion)](characters/Dantalion.md) — 伊利亞二王子，「倒吊人」的實際控制者。
- [伊南娜 (Inanna)](characters/Inanna.md) — 伊利亞王女，象徵著國家的希望。
- [歌洛莉亞 (Gloria)](characters/Gloria.md) — 騎士同盟「聖槍騎士」，正義感的化身。
- [薩曼莎 (Samantha)](characters/Samantha.md) — 法皇國高階祭司，追求公義與信仰。
- [奧古斯特 (Auguste)](characters/Auguste.md) — 騎士同盟的將軍，極端的政治現實主義者。
- [卡里斯 (Caris)](characters/Caris.md) — 羅迪尼亞法皇國高階樞機卿，極權宗教的代表。
- [索菲亞 (Sofia)](characters.md#索菲亞-sofia) — 伊利亞王國特務機構「倒吊人」的首領。
- [馬格努斯 (Magnus)](characters.md#馬格努斯-magnus) — 伊利亞王國軍將領，剛毅的老兵。
- [貝瑞爾 (Beryl)](characters.md#貝瑞爾-beryl) — 天才晶石魔法師。
- [科爾 (Col)](characters.md#科爾-col) — 法皇國刺客，沈默寡言的暗殺者。
- [諾諾薇兒 (Nonowill)](characters.md#諾諾薇兒-nonowill) — 薇爾德部落的核心人物，歌者。
- [莉莉薇兒 (Lilywill)](characters.md#莉莉薇兒-lilywill) — 薇爾德戰士，擅長遠程狙擊。
- [阿加塔 (Agata)](characters.md#阿加塔-agata) — 北方冰原線的關鍵引導者。
- [路特菲 (Lutfi)](characters.md#路特菲-lutfi) — 騎士同盟成員，與歌洛莉亞有複雜關係。

- [浪湧城](factions.md#浪湧城) — 伊利亞的軍事重鎮與戰略要地。
