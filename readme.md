# GamePlots - 敘事研究與創作參考庫

**GamePlots** 是一個旨在系統化收集、分析並轉化各類媒介（遊戲、動漫、小說、影視）敘事精華的專案。透過 Agent 的協助，我們將龐雜的網路資料轉化為結構化的故事元素庫，作為原創遊戲設計、世界觀建構與角色創作的深度參考。

---

## 🚀 專案願景

我們相信優秀的故事具備可分析的模式與迷人的元素。本專案的目的在於：
- **跨媒介收集**：廣泛攝取遊戲 wiki、攻略、劇情心得及角色解析。
- **深度分析**：利用 Agent 提取核心衝突、角色成長曲線及獨特的世界觀設定。
- **創作轉化**：將分析結果分類彙整，為新的創意提供靈感來源與素材參考。

---

## 📂 專案結構

本專案採用嚴謹的目錄管理，確保從原始資料到最終產出的流程清晰可見：

- **`working/`**：各作品的活動工作區。
    - `working/<作品名稱>/raw/`：存放未經加工的原始素材（Wiki 備份、文本等）。
    - 存放分析過程中的暫存檔與初步報告。
- **`results/`**：最終資料庫。存放已整理完畢、結構清晰的作品分析報告。
- **`classes/`**：主題彙整區。依據背景（如：奇幻、科幻）或角色原型（如：傲嬌、史詩英雄）分類的跨作品精選集。
- **`scripts/`**：工具箱。存放自動化爬蟲、文本解析器等。
- **`skills/`** & **`skills_index.md`**：提示詞 (Prompt) 資源庫。
- **`for_agent.md`**：**Agent 執行規範**（所有 Agent 動工前的必讀文件）。

---

## 🛠️ 技術與安全性

- **Agent 導向**：主要任務由 AI Agents 執行，確保資料處理的大規模與系統化。
- **安全性優先**：在進行網路爬取與腳本執行時，必須嚴格遵守安全規範與 `for_agent.md` 中的規則。
- **跨平台支持**：考慮到 Windows 與 Linux 環境的切換，所有工具與指令均需具備環境相容性。

---

## 📜 核心鐵則 (摘要)

1. **語言**：本專案對象為繁體中文使用者，所有產出一律使用**繁體中文**。
2. **優先閱讀**：執行任務前必須讀取 `for_agent.md`。
3. **資料隔離**：原始素材與分析結果必須明確區隔。
